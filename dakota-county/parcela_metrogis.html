Section 5 Attributes
Overview A standard set of attribute fields is included for each county. The attributes are identical for the point and polygon datasets. Not all attributes fields are populated by each county.

A list of the attribute names is shown below. Detailed information about the attributes can be found at:
http://www.datafinder.org/metadata/MetroGIS_Regional_Parcels_Attributes.pdf

Note: The last five digits of Dakota Counties PIN Numbers changed format in March of 2011
See .pdf above for further details.

A lookup table "ParcelsDakotaOldNewPINLookupTable2011.dbf" will be included in the Dakota County zip bundle that allows users to join parcel information which contain the old PIN format. Users should be aware that the old and new PIN formats are not mutually exclusive to each other. There are approximately 1,500 records of NEWPIN numbers that have the same PIN as OLDPIN number.
